## Running the App

### iOS Simulator
```bash
npx expo start --ios -c
```

### Android Emulator
```bash
npx expo start --android -c
```

The `-c` flag clears the Metro cache for a clean start. Requires Xcode (iOS) or Android Studio (Android) to be installed.
